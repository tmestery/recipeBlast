# recipeBlast

**INSTANTLY understand what's ACTUALLY in your food.**

A full-stack application that analyzes recipes and food items, providing nutrition breakdowns, warnings, and insights using frontend and backend integrations—including a **local LLM**, REST APIs, OCR, and database management.

**Tools / IDEs:** VSCode, IntelliJ, GitHub

## Installation and Usage

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

* Open the backend project in your IDE and run `BackendApplication`.

### Local LLM Server

* Required for local recipe and health tip generation:

```bash
pip install ollama
ollama serve
```

### Tesseract (OCR)

* Used for image-to-text parsing:

```bash
brew install tesseract
```

## Features

* User login/signup system with frontend forms and backend validation
* Recipe and food item upload & management
* Ingredient parsing and database storage
* Nutrition analysis and health warning flags
* Alternative product suggestions
* Daily health tips generated by a **local LLM**
* Secure REST API backend with Spring Boot
* Responsive UI built with React
